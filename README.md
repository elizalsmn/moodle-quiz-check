# moodle-quiz-check
new quiz drop on moodle, i hope it works for other website as well

## Setting up
1. create venv
   ```
   python3 -m venv .venv
   source .venv/bin/activate
   ```
2. install extensions
   ```
   pip install playwright python-dotenv
   playwright install
   ```
3. run the code
   ```{python}
   python main.py
   ```
